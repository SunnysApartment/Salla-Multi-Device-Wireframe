## ✨ Overview

**Salla Multi-Device Wireframe & Bento Grid Simulator** is a professional interactive design tool built specifically for **Salla Stores**.

The simulator enables designers and developers to create modern **Bento Grid banner layouts**, preview them in real-time across desktop and mobile devices, customize content visually, and export clean production-ready code compatible with Salla's custom HTML sections.

Whether you're building landing pages, promotional sections, campaign banners, or homepage experiences, this tool dramatically speeds up the design-to-deployment workflow.

## 🛍️ Salla Integration Guide

Deploying your design to Salla takes only a few minutes.

### Step 1

Login to your Salla Dashboard.

### Step 2

Navigate to:

```text
Page Builder
or
Code Editor
```

### Step 3

Create a new:

```html
HTML Component
```

### Step 4

Customize your Bento Grid inside the simulator.

Update:

* Images
* Text Content
* Buttons
* Colors

### Step 5

Open:

```text
Export Salla Code
```

and click:

```text
Copy Code
```

### Step 6

Paste the generated code into Salla and save.

🎉 Your Bento Grid is now live.
